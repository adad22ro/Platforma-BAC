# Componente UI

> Actualizat la: 2026-06-26

## Cum sunt organizate

Componentele React reutilizabile se află în `/components`. Fiecare componentă va fi documentată aici după ce e creată.

### Format standard

```
### NumeComponenta
Fișier: components/NumeComponenta.tsx
Scop: ce face această componentă

Props:
| Prop | Tip | Obligatoriu | Descriere |
|---|---|---|---|
| ... | ... | ... | ... |

Unde e folosită: lista paginilor/componentelor care o folosesc
```
